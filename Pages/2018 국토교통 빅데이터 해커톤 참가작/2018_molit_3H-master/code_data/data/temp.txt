데이터로 부터 어떤걸 보여줘야 하는지 그걸로 어떤가설  ==> 어떤정책을 피는게 맞는지 

1. 데이터로 부터 어떤걸 보여주는지 
- 발견
전문직수와 공시지가수가 젠트리피케이션 과 밀접한 관계가 있음을 데이터 분석하는 과정에서 알수 있었음 
(** 전문직수,공시지가수 - 젠트리피케이션  관계 차트 보여주면 좋을것 같음 **)

- 아이디어
전문직수와 공시지가수를 예측한다면 젠트리피케이션을 예측할 수 있을거라는 아이디어를 얻음

- 예측 방법 설정
머신러닝 기법중 하나인 딥러닝과 lstml을 사용하여 전문직수와 공시지가수를 예측하는 모델링을 작성함
input데이터 로는 지역(지역은 동별로 분류)별로 연령, 총사업체수, 총인구수, 전입전출 수와 크롤링을 통하여 기사화된 수(이슈거리 여부)를 이용하고 input 데이터는 전문직수, 공시지가수 이다.
(** input데이터와 ouput 데이터의 상관 관계도 차트로 **)
(** 결국 input 데이터로 젠트리피케이션을 얼마나 의미있게 유추했는지를 차트로 **)


2. 가설 : 위의 모델링으로 의미있는 값을 얻어 왔을경우만을 생각
젠트리피케이션은 아래와 같은 변수에 차이가 있을 수 있다(각각이 원자값이 아닌 경우 포함)
- 이슈화된 수(뉴스기사에서의 언급 수)
- 평균 연령대
- 총사업체수
- 총인구수
- 전입 전출 
==> 또한 전문직수 와 공시지가수가 많아지면 위 변수가 차이가 있다.


3. 어떤정책을 피는게 맞는지
 - 보증금 대출 제도
젠트리피케이션으로 인하여 피해보는 사람들(소상공인 임대자) 특히 지역적 문화가 있는경우 이를 보호 하기 위하여 해당 증상으로 인하여 증가하는 보증금 만큼을 지원하여 보호함
(** 젠트리케이션으로 인하여 집값이 상승한구역과 구역별 얼마나 몇퍼센트 상승했는지 보여주는 차트**)

 - 이미 있는 제도의 강화
젠트리피케이션으로부터 소상공인을 보하하기 위하여 젠트리피케이션이 일어난 경우 건물을 국가에서 임대 및 매입하여 제공해주는 법률.
기존 젠트리피케이션으로 인하여 피해를 보고 많은 지역적 특성이 사라진 후 일부를 지키기 위한 것이 아닌, 기존 방식보다 빠른 젠트리피케이션현상을 인지 함으로써 지역적 특성을 보호할 수 있음.

- 부동산 안정화 ( ? 이건... 음... 할지말지... )
젠트리피케이션으로 인하여 집값이 급상승하는 만큼 그에 따른 부동산불안정화를 국가의 개입(대기업의 진입을 제한적으로 허용 방식, 음식 체인점 부분 허가 방식, 매매가 동결 방식<이부분 생략 가능 ..>)으로 보호할 수 있음
