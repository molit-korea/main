Installing the GeoServer GeoPackage extension
=============================================

 #. If you haven't done already, install the WPS extension: :ref:`wps_install`.

 #. Download the Geopkg extension from the `nightly GeoServer community module builds <https://build.geoserver.org/geoserver/master/community-latest/>`_.

    .. warning:: Make sure to match the version of the extension to the version of the GeoServer instance!

 #. Extract the contents of the archive into the ``WEB-INF/lib`` directory of the GeoServer installation.
