GeoServer 2.9+ Teradata Extension
---------------------------------

This package contains a Teradata DataStore implementation that is 
distributed as a separate plug-in. 

Please report any bugs with jira (https://osgeo-org.atlassian.net/projects/GEOS).

Any other issues can be discussed on the mailing list (http://lists.sourceforge.net/lists/listinfo/geoserver-users).

Installation
------------

1. Copy gt2-jdbc-teradata jar into the WEB-INF/lib directory of your GeoServer
   installation.

2. Download the Teradata jdbc driver and unpack into WEB-INF/lib: 

     https://downloads.teradata.com/download/connectivity/jdbc-driver

3. Restart GeoServer.
